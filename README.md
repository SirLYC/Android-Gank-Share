### 简介
- 分享每日技术干货
- 精选每日妹子图
- 还有休息视频

### 版本说明
#### V2.1更新（2017.7.11）
- （题外话）总算熬过考试周了
- 侧滑背景改用bing每日一图（郭霖大神的api）
- 网络请求框架改为rxjava+retrofit
- 修复acytivity后fragment内存泄漏的bug
- 监听网络、时间状态，进行内容刷新
- 优化代码（强迫症看着更舒服吧）
---
#### V2.0更新（2017.6.13）
- 重新排版界面，增加每日推荐（今天没更新会加载前一天）
- 优化android 5.0 以上版本的界面显示

---
#### V1.0
- 基本的展示文章，阅读
- 支持保存妹子图到本地
- 支持干货分享（QQ）
- 搜索功能没写（逃）
- ...
